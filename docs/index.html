For AI Agents

Add this to your agent's instructions (CLAUDE.md, AGENTS.md, system prompt):

## Reminders
You have access to `agentrem` CLI for persistent reminders across sessions.

On session start: agentrem check --type time,session --budget 800
When user says "remind me": agentrem add "<content>" --due "<when>" --priority <1-5>
Key commands: add, check, list, search, complete, snooze
